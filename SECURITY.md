# Security Policy

## Reporting a vulnerability

If you discover a security issue in Disclosure Alpha, please report it responsibly:

1. **Preferred:** [GitHub Security Advisories](https://github.com/alwank/disclosure-alpha/security/advisories/new) (private disclosure)
2. **Alternative:** Email the maintainer via the contact on [PyPI](https://pypi.org/project/disclosure-alpha/)

Please do not open public GitHub issues for undisclosed security vulnerabilities.

We will acknowledge receipt and work on a fix. Coordinated disclosure is appreciated.

## Scope

This policy covers the open-source codebase (CLI, Python SDK, HTTP API, MCP servers) in this repository. It does not cover third-party services (SEC EDGAR, market data providers) or deployments you run yourself.

## Not investment advice

Disclosure Alpha produces deterministic analytics from SEC filing text. Output is for research and integration testing — not investment, legal, or accounting advice.

Full scope, validation limits, and legal disclaimers: [Scope and claims](https://disclosure-alpha.readthedocs.io/en/latest/getting-started/scope-and-claims.html) · [Legal](https://disclosure-alpha.readthedocs.io/en/latest/legal.html)
